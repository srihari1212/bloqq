Different Models

  • First Model-COST function
    • In the first model, equal preference is given to all the choices.
    • Every variable has values between 1-10, default value is set to 5.
  • Second Model - ARIMA
    • Priority is assigned by setting the higher values to that variable.
    • You have 30 points which can be distributed equally or as per the choice. The summation of all the variables should make the count of 30.
    • Added Cost, storage and days input variables

      Given the storage(required) and number of days you may compute the cost

      Given the budget and storage(required) you may compute days (you can buy the storage)
